M.O.R.Z. Purge

Development Time:

September 2025 - Current

Summary:

M.O.R.Z. Purge - the sequel to Project M.O.R.Z. - is a first-person shooter that pins you against aggressive alien insectoids. In the abandoned wastelands left by the people before you, you can find ammo, new guns, and weapon upgrades. Make your way through the cave system to fight the Broodmother, and put an end to the alien invasion.

About the Project:

This game is being created by myself and six other people. We are making it as part of a semester-long assignment for out CIT324 class.

M.O.R.Z. Purge is being made using Unity 6, which uses C# scripts.

Our team uses agile development techniques, such as scum practices.

This project utilizes a branched GitHub enviroment for hosting each member's work as well as verison control.

Contributions:

  • Enemy superclass script
  • Melee and flying enemy scripts
  • Enemy sprites
  • Enviromental textures
  • Pick-up and enemy spawn management
  • Assisted with prop implementation
  • Served as Scrum master for the second sprint

BugFishing

Development Time:

January 2025 - April 2025

Summary:

BugFishing is a fishing simulator, but with bugs! Play as a frog making a living by fishing up bugs, selling your catches, and then upgrading your gear. Discover different species, and keep track of your learnings in your notebook.

About the Project:

This game was made as an entirely solo project for my CIT216 class.

BugFishing was developed using Unity 2022.3.24f1, which uses C# scripting.

The development utilized GitHub version control.

Contributions:

I created everything other than sound effects and music. This includes:

  • Game and inventory management scripts
  • Scripts for mouse-responsive, randomised system for casting, reeling, and catching bugs
  • Notebook encyclopedia scripts
  • Insect 3D models
  • UI sprites and scripts
